# Fundamentals of Intelligent Systems

This repository contains projects, assignments, and implementations for the "Fundamentals of Intelligent Systems" course. The goal of this course is to explore the core concepts and algorithms in artificial intelligence and machine learning.

This repository covers a wide range of topics, from introductory AI concepts to advanced optimization techniques. Below is a summary of the key areas explored in this course:

### 1. Introduction to Artificial Intelligence
-   Definitions and applications in electrical engineering and other fields.
-   Core tasks: Classification, Recognition, Prediction, Clustering, Regression.
-   Introduction to Multi-Layer Perceptron (MLP) and Gradient Descent.

### 2. Dimensionality Reduction
-   Linear Methods:
    -   Principal Component Analysis (PCA)
    -   Linear Discriminant Analysis (LDA)
-   Introduction to Non-linear Dimensionality Reduction techniques.

### 3. Support Vector Machine (SVM)
-   Linear classifiers and the concept of maximum margin.
-   Support Vectors, Soft Margin, and the Kernel Trick.

### 4. RBF Neural Networks
-   Definition, architecture, and applications.
-   Supervised and unsupervised training approaches.

### 5. Decision Trees
-   Fundamentals of decision trees: nodes, branches, and leaves.
-   Algorithms: ID3, Entropy calculation.
-   Advantages, disadvantages, and tree pruning techniques.

### 6. Reinforcement Learning (RL)
-   Core concepts and characteristics of RL.
-   Introduction to the Q-Learning algorithm.

### 7. Clustering
-   Definitions, applications, and the distinction from classification.
-   Data types and feature selection for clustering.
-   Cluster evaluation metrics (Intra-cluster and Inter-cluster validation).
-   Algorithms:
    -   Hierarchical Clustering
    -   Centroid-based Clustering (e.g., K-Means)
    -   Density-based Clustering (e.g., DBSCAN)

### 8. Optimization
-   Fundamental concepts of optimization problems (objective function, search space).
-   Constraints and multi-objective optimization.
-   Evolutionary Algorithms:
    -   Genetic Algorithms (GA)
    -   Particle Swarm Optimization (PSO)
-   Population-based methods and constraint handling techniques.
